# Kill Team Viewer

Offline field-reference tool for Kill Team — faction rules, operative datacards, weapon profiles, abilities, equipment and FAQ for quick lookup during games.

**Live:** https://soucemustache.github.io/KT/killteam_viewer.html

## Features

- Single-file offline PWA (works without a connection via Service Worker)
- Side-by-side reference for your team and your opponent's
- Operative selection and equipment tracking, saved between sessions
- Weapon type icons (ranged / melee)
- Installable to your phone's home screen

## Tech

Vanilla JS, zero dependencies, deployed via GitHub Pages.

Game data sourced from Wahapedia.
